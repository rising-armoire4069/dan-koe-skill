## 💡 Troubleshooting common issues

If the application does not open, check these points.

* Restart your computer. Sometimes background processes block new installs.
* Check your antivirus software. Sometimes security settings flag new applications. Create an exception for the installation folder if needed.
* Ensure you have the latest Windows updates installed. Older versions of Windows may lack the necessary support libraries.
* If the app feels slow, close other programs like web browsers. Large AI models require memory to run well.
